# 색상 배치 레이아웃

---

## 앱 설명

- 상단 AppBar에는 **"색상 배치 레이아웃"** 표시
- 300x300 박스 안에 다양한 색상의 박스를 배치한 레이아웃 구성
- 상단은 빨강/파랑/검정/주황색, 하단은 노란색 박스로 나뉨
- `Expanded`, `Row`, `Column`, `Container` 위젯을 활용하여 레이아웃 구성

---

## 전체 코드

```dart
import 'package:flutter/material.dart';

void main() {
  runApp(const MyApp());
}

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: '색상 배치 레이아웃',
      theme: ThemeData(
        primarySwatch: Colors.blue,
      ),
      home: const MyHomePage(),
    );
  }
}

class MyHomePage extends StatelessWidget {
  const MyHomePage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text('색상 배치 레이아웃'),
      ),
      body: Center(
        child: Container(
          width: 300,
          height: 300,
          child: Column(
            children: [
              // 상단 부분
              Expanded(
                flex: 1,
                child: Row(
                  children: [
                    // 왼쪽 빨간색 박스
                    Expanded(
                      flex: 1,
                      child: Container(
                        color: Colors.red,
                      ),
                    ),
                    // 오른쪽 부분
                    Expanded(
                      flex: 1,
                      child: Column(
                        children: [
                          // 파란색 박스
                          Expanded(
                            flex: 1,
                            child: Container(
                              color: Color(0xFF0000FF), // 더 진한 파란색
                            ),
                          ),
                          // 검정색과 주황색 행
                          Expanded(
                            flex: 1,
                            child: Row(
                              children: [
                                // 검정색 박스
                                Expanded(
                                  flex: 1,
                                  child: Container(
                                    color: Colors.black,
                                  ),
                                ),
                                // 주황색 박스
                                Expanded(
                                  flex: 1,
                                  child: Container(
                                    color: Colors.orange,
                                  ),
                                ),
                              ],
                            ),
                          ),
                        ],
                      ),
                    ),
                  ],
                ),
              ),
              // 하단 노란색 부분
              Expanded(
                flex: 1,
                child: Container(
                  color: Colors.yellow,
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}
```
